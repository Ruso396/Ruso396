## 🐍 Contribution Snake

<div align="center">

  <source media="(prefers-color-scheme: light)" srcset="https://raw.githubusercontent.com/Ruso396/Ruso396/output/github-snake.svg" />

</div>

> 🛠️ **One-time setup required.** The snake eats your real contribution graph,
> so it has to be generated *inside your own* `Ruso396/Ruso396` profile
> repository via GitHub Actions — I can't render it for you from outside
> your account. It takes about two minutes:
>
> 1. In your `Ruso396/Ruso396` repo, create `.github/workflows/snake.yml`
> 2. Paste the workflow below and commit it
> 3. Go to the **Actions** tab and run it once manually — after that it
>    re-runs automatically every day at midnight UTC

```yaml
name: generate snake animation

on:
  schedule:
    - cron: "0 0 * * *"          # runs daily at 00:00 UTC
  workflow_dispatch: {}           # lets you trigger it manually
  push:
    branches:
      - main

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    steps:
      - name: Generate snake animation SVG
        uses: Platane/snk@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark

      - name: Push snake output to the "output" branch
        uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
